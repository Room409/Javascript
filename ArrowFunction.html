Arrow Function

Que es un arrow Function?

Una arrow function (función flecha) en JavaScript es una forma más concisa y simplificada de escribir una función. Se caracteriza por utilizar la sintaxis de una flecha (=>) para definir la función.

Sintaxis de una Arrow Function

const saludar = nombre => console.log(`¡Hola, ${nombre}!`); saludar("Juan"); // Output: ¡Hola, Juan!

Ventajas de usar Arrow Function

Sintaxis mas concisa, Compatibilidad con funciones de alto orden: Las arrow functions son especialmente útiles cuando se utilizan en funciones de alto orden, como map, filter y reduce, ya que permiten escribir un código más limpio y declarativo.
