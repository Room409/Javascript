Destructuring

A que se refiere con Destructuring?

Es una característica introducida en ECMAScript 6 (ES6) de JavaScript que permite extraer valores de objetos o arreglos y asignarlos a variables individuales de manera más concisa. En lugar de acceder a cada propiedad de un objeto o elemento de un arreglo mediante la notación de punto o corchetes, el Destructuring proporciona una sintaxis más compacta para extraer los valores directamente en variables separadas.

Ejemplo de uso

                        const persona = { nombre: 'John', edad: 30 };

                        const { nombre, edad } = persona;

                        console.log(nombre); // Salida: John
                        console.log(edad); // Salida: 30
