Parametro ...rest

Que es ...rest?

El operador de rest (...rest) en JavaScript se utiliza para trabajar con un número variable de elementos en arreglos u objetos, ya sea desestructurándolos o creando nuevos objetos con propiedades adicionales.

Para que se usa ...rest?

se utiliza para desestructurar arreglos o objetos, y permite trabajar con un número variable de elementos. Este operador puede tener diferentes usos dependiendo del contexto.

Ejemplo de uso

function sumar(...numeros) { //Tomamos parametro rest
                    let resultado = 0;
                    for (let i = 0; i < numeros.length; i++) {
                      resultado += numeros[i];
                    }
                    return resultado;
                  }
                  
                  console.log(sumar(1, 2, 3, 4, 5)); // Resultado: 15
                  console.log(sumar(10, 20, 30)); // Resultado: 60
