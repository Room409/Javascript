Variable Let y Const

Que es una variable let?

Permite declarar variables con alcance de bloque y su valor puede ser actualizado y reasignado, se utiliza cuando se necesita una variable mutable (cambiable), cuyo valor puede cambiar a lo largo del programa

Que es una variable Const?

También permite declarar variables con alcance de bloque, pero su valor no puede ser reasignado después de su declaración inicial, lo que las convierte en constantes, se utiliza para declarar constantes inmutables, cuyo valor no debe cambiar después de su inicialización.
