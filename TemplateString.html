Template String

Que es un TemplateString

Los Template Strings se definen utilizando comillas invertidas (` `) en lugar de comillas simples o dobles utilizadas en las cadenas de texto convencionales. Dentro de un Template String, se pueden incluir expresiones o variables utilizando la sintaxis ${variable}. Estas expresiones o variables son evaluadas y su resultado es insertado en la cadena de texto.

Ejemplo de uso

                        const name = 'John';
                            const age = 30;

                            const message = `Hola, mi nombre es ${name} y tengo ${age} años.`;

                            console.log(message);
                            // Salida: Hola, mi nombre es John y tengo 30 años.
